# Introduction

Trackion is a lightweight telemetry infrastructure tool designed for developers who need **real-time analytics** and **custom event tracking** without the complexity of enterprise solutions.

## What is Trackion?

Trackion helps you understand what's happening in your applications by providing:

- **📈 Real-time Analytics** - Track page views, user sessions, and engagement metrics
- **🎯 Custom Events** - Monitor specific actions, workflows, and business metrics
- **🔒 Privacy-First** - Self-hosted solution with complete data ownership
- **⚡ Quick Setup** - Start tracking in under 2 minutes with a simple script tag

## Key Features

### Analytics Tracking

Monitor your website or application with automatic page view tracking, session management, and user engagement metrics. Get insights into how users interact with your application.

### Custom Events

Track any action or workflow in your application using a simple REST API. From user signups to complex business processes - if it's important to your business, you can track it.

### Real-time Dashboard

View all your metrics in a clean, minimal dashboard inspired by tools like Pi-hole. See what's happening in your application right now, not hours later.

### Developer Experience

- Simple integration with one script tag
- RESTful API for custom events
- Comprehensive documentation
- No vendor lock-in

## Current Status

::: warning Note
**Workflows and Pipelines** are not yet supported. Trackion currently focuses on analytics tracking and custom events. Advanced workflow orchestration features are planned for future releases.
:::

## What's Different?

Unlike other analytics solutions, Trackion is:

- **Self-hosted** - Your data stays on your servers
- **Developer-focused** - Built by developers, for developers
- **Minimal** - No bloated features you don't need
- **Fast** - Lightweight tracking with minimal overhead
- **Open Source** - MIT licensed, completely transparent

## Use Cases

- **Web Analytics** - Track page views, user sessions, and engagement
- **Application Monitoring** - Monitor key user actions and flows
- **Business Metrics** - Track conversions, signups, and custom KPIs
- **Performance Insights** - Understand how users interact with your app
